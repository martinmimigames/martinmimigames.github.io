little keyboard

This is a little keyboard for android.

The total size is less than 20Kb, to minimize impact on your storage.

The app is not completed yet, however basic function works.

Features includes:

- ESC and CTRL button support*

- Volume button as left right arrow

- Special key to allow unicode composing (lower right)

- Basic keyboard

Symbol layout is not developed yet...

*not fully tested

To enable:

1. Go to Settings

2. Find Language and Input

3. Find Manage Keyboards

4. Enable Little Keyboard
